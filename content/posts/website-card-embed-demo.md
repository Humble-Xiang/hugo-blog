#### 使用 Markdown 引用网站

> [what is hugo](https://gohugo.io/about/what-is-hugo/)

#### 通过 iframe 使用 Website Card Embed 引用网站

<iframe src="https://website-card-embed.vercel.app/?url=https://gohugo.io/about/what-is-hugo/" style="width:100%;height:124px" frameborder="no"></iframe>

#### 通过 Hugo Custom Shortcode 使用 Website Card Embed 引用网站

{{< card "https://gohugo.io/about/what-is-hugo/" >}}
